Repositories of code I have written to accompany my research papers.

  • View all on GitHub

  • Cloudy-Maraston

    Jupyter notebooks reproducing all plots in Newman et al. (2025, MNRAS). Uses Python and the synthesizer package to generate synthetic astrophysical observables from stellar population model grids.

    • Repository
    • Paper

  • microlensing-in-M31

    Code and data to reproduce all results in Newman, Middleton & McMaster (2025). Models microlensing by white dwarfs to resolve accretion flows in M31.

    • Repository
    • Paper

  • synthesizer

    A Python package for generating synthetic astrophysical observables from galaxy and black hole physics models. I have contributed code to this open-source project, including integration with the Cloudy-Maraston grids.

    • Docs
